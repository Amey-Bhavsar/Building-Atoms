2. Bubble Sort - O(n^2)
  Array gets sorted from the last element. First for loop = arr.size() - 1. Basically n-1 loops
  And in second for loop you swap each adjacent elements for loop bounds : j = 0 to j = arr.size() -1 -i
  Sample code from gfg : https://www.geeksforgeeks.org/problems/bubble-sort/1
